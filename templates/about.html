About VisionAIHub

VisionAIHub is a modular AI web application that combines computer vision, classical machine learning, transfer learning, and a Gemini-powered assistant to analyze road and outdoor images.

PROJECT GOAL

One image. Multiple AI insights.

The platform replaces separate tools with one connected workflow for object detection, traffic analysis, weather recognition, scene understanding, and natural-language explanation.

CORE MODULES

How the platform analyzes an image

Each module performs a different technical role.

01

YOLOv8 Detection

Detects and counts people, vehicles, traffic lights, stop signs, and other supported road objects.

02

Feature Engineering

Converts YOLO detections into 25 numerical features, including density, occupancy, ratios, and confidence statistics.

03

Traffic ML

A Random Forest classifier predicts Low, Medium, or High traffic density from the engineered features.

04

Weather Classification

EfficientNetB0 classifies the image as Cloudy, Foggy, Rainy, or Sunny.

05

Scene Classification

A transfer-learning model classifies the environment as Buildings, Forest, Glacier, Mountain, Sea, or Street.

06

Gemini Assistant

Converts structured model outputs into grounded explanations and answers user questions about the analysis.
